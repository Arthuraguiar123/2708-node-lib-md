A interface File provê informações sobre arquivos e permite ao JavaScript  a acessar seu conteúdo.

São geralmente recuperados a partir de um objeto [FileList](https://www.playstation.com/pt-br/games/god-of-war/) que é retornado como resultado da seleção, pelo usuário, de arquivos através do elemento [<input>](https://store.steampowered.com/app/1593500/God_of_War/), a partir do objeto [DataTransfer](https://www.youtube.com/watch?v=W0BDB9DPbWE) utilizado em operações de arrastar e soltar, ou a partir da API `mozGetAsFile()` em um [HTMLCanvasElement](https://zany-giggle-wrj4g6j5qjg4f5gq9.github.dev/). Em Gecko, códigos com privilégiios podem criar objetos File representando qualquer arquivo local sem a intereção do usuário (veja [Implementation notes](https://classroom.google.com/w/ODQyNDkxNjEwMDIx/t/all) para mais informações.).
